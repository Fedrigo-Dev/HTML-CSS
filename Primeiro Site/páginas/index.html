Linguagens de programação? 🔥

O que são?

Imagine que você está tentando explicar para um amigo estrangeiro como fazer um café,
mas vocês não falam a mesma língua. Você precisaria de um tradutor ou de um conjunto
de gestos e palavras que ambos entendam para que o café saia no ponto certo. Uma
linguagem de programação é exatamente isso: um conjunto de regras, símbolos e
palavras usado para que nós, humanos, possamos dar instruções precisas a um computador.

Imagem de Bits de Computador

Como elas funcionam?

Para entender como elas funcionam, imagine que você é um arquiteto enviando instruções
para uma equipe de construção que só entende comandos matemáticos puros.
O processo é uma verdadeira esteira de tradução

A Escrita (Código-Fonte)

Tudo começa com você escrevendo um arquivo de texto. Você usa uma sintaxe específica (regras de gramática da linguagem).
Exemplo em Python: ("Olá, mundo!")
Esse comando é fácil para você ler, mas o processador do computador não tem a menor ideia do que "print" significa.

A Tradução (O Coração do Processo)

Como o computador só entende impulsos elétricos (0 e 1),
precisamos de um tradutor. Existem dois tipos principais:


Compiladores: Eles leem todo o seu código de uma vez e o transformam
em um arquivo executável (como um .exe). É como traduzir um livro
inteiro antes de entregá-lo ao leitor. (Ex: C++, Rust).

Interpretadores: Eles traduzem e executam o código linha por linha,
em tempo real. É como um tradutor simultâneo em uma palestra.
(Ex: Python, JavaScript).
